jmeter-rmi-plugin by jmibanez

Plugin to add support for recording and playing back RMI calls

This is a self-contained plugin to JMeter that allows one to record and play back RMI calls. To build, you will need an existing JMeter installation and Ant.

Dependencies

JMeter 2.3 or later, Ant
